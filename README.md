## Projet 1 : site de presentation d'un restaurent
## description 
ce projet consiste a creer un lending page complete pour un restaurent en utilisant HTML, CSS, Bootstrap et Javascript.

## fONCTIONNALITES A VENIR

 *Menue du restaurent en pop-up.
 *Section de reservation de table .
